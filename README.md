# Jackon's Snay.io Script

A custom mod menu I made for Snay.io. It comes with an FPS unlocker, visual quality settings, macro/combo sequences, and RGB themes.

![Menu Preview](https://raw.githubusercontent.com/Jackon0234/Assest/main/menugorunum.png)

## How to Install

Since this is an unpacked browser extension, it's pretty easy to load into Chrome, Edge, or Brave:

1. Click the green **`Code`** button at the top right of this page and hit **`Download ZIP`**.
2. Extract the ZIP file somewhere on your PC.
3. Open your browser and head over to your extensions page:
   - Chrome: `chrome://extensions/`
   - Edge: `opera://extensions/`
4. Turn on **"Developer mode"** (usually in the top right corner).
5. Click the **"Load unpacked"** button.
6. Select the folder you just extracted (the one containing `manifest.json`).
7. Jump into [Snay.io](https://snay.io/) and you should see the menu pop up on the left side!

Have fun!
